# Vedansh Arora — Portfolio

A Next.js portfolio site with four pages: Home, Work, Expertise, and Contact.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:4000](http://localhost:4000) in your browser.

## Pages

- `/` — Home with hero, expertise bento grid, projects, experience, and CTA
- `/work` — Selected work case studies
- `/expertise` — About, skills, and experience timeline
- `/contact` — Contact form and social links

## Tech Stack

- Next.js 15 (App Router)
- React 19
- Tailwind CSS
- TypeScript
